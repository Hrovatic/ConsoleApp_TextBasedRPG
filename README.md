# TextBasedRpg

A console-based RPG built with C# and .NET.

## Features

- Character creation
- Turn-based combat
- Random encounters
- Currently infinite gameplay till character death

## Tech Stack

- C#
- .NET Console App

## Run

From repository root:

```bash
dotnet run --project TextBasedRpg/TextBasedRpg.csproj
```

Or from inside the project folder:

```bash
dotnet run
```
